# IT & Cybersecurity Resource Hub

## What Is This?

This is a beginner-friendly educational website designed to teach foundational concepts in information technology and cybersecurity. The site includes multiple pages covering IT fundamentals, cybersecurity basics, and certification pathways. It is structured to be simple, organized, and easy to navigate for new learners while also introducing more advanced concepts in a manageable way.

## Why Does It Exist?

This project exists to provide an accessible learning resource for students, entry-level IT learners, and individuals transitioning into technology careers. Many technical topics can feel overwhelming at first, so this site simplifies key concepts and presents them in a structured and beginner-friendly format.

The project also serves as a practical portfolio project that demonstrates web development skills, problem-solving, organization, and the ability to build and improve a deployed website over time.

## What Tools Did I Use?

- **Visual Studio Code** – Used as the primary development environment for writing and editing code.
- **HTML** – Used to structure the content across all pages.
- **CSS** – Used to style the layout, improve visual design, and create responsive page designs.
- **JavaScript** – Used to implement interactivity such as dark mode, search functionality, and filtering features.
- **GitHub Desktop** – Used for version control, commits, and managing project changes.
- **GitHub Pages** – Used to deploy and host the live website.
- **ChatGPT** – Used for troubleshooting, refining features, improving styling, and assisting development workflow.
