# IT & Cybersecurity Resource Hub

## What Is This?

This is a beginner-friendly educational website designed to teach foundational concepts in information technology and cybersecurity. The site includes multiple pages covering IT fundamentals, cybersecurity basics, and certification pathways. It is structured to be simple, organized, and easy to navigate for new learners.

## Why Does It Exist?

This project exists to provide an accessible learning resource for students, entry-level IT learners, and individuals transitioning into technology careers. Many technical topics can feel overwhelming, so this site simplifies key concepts and presents them in a structured way. It also serves as a practical portfolio project demonstrating web development skills.

## What Tools Did I Use?

- **Visual Studio Code** – Used as the primary development environment for writing and editing code.
- **HTML** – Used to structure the content across all pages.
- **CSS** – Used to style the layout, improve visual design, and create responsiveness.
- **JavaScript** – Used to implement interactivity such as dark mode, search functionality, and filtering features.
- **GitHub Desktop** – Used for version control, commits, and managing changes.
- **GitHub Pages** – Used to deploy and host the live website.
- **ChatGPT** – Used for troubleshooting, refining features, and improving development workflow.
